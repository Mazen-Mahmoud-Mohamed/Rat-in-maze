# Rat-in-maze-
Overview

A maze-solving project that finds paths from the start point to the destination using different pathfinding techniques and algorithms.

Features
Multiple maze-solving approaches
Path visualization
Efficient route searching
Grid-based maze navigation
Technologies Used
Python
NumPy
Queue/Deque Algorithms
Algorithms Implemented
Breadth-First Search (BFS)
Depth-First Search (DFS)
Recursive Backtracking
